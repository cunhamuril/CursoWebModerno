## Angular

#### 1- Instalar globalmente

```bash
npm i -g @angular/cli

# or

yarn global add @angular/cli
```

#### 2- Criar projeto

```bash
ng new front-angular --minimal
```

#### 3- Iniciar projeto

```bash
npm start

#or

yarn start
```

#### 4- Criar componente

```bash
ng g c views/home

# or

ng generate component views/home
```

## React

#### 1- Criar projeto


## Vue

#### 1- Instalar globalmente
